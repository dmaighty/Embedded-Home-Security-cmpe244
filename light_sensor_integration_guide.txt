# **INTEGRATION GUIDE (SHORT)**

1. **Add files to the project**

   * Add `LightSensor.c`
   * Add `LightSensor.h`

2. **Include the module**

   ```c
   #include "LightSensor.h"
   ```

3. **Start the light sensor + buzzer tasks**
   Call this inside `main()` **before** `vTaskStartScheduler()`:

   ```c
   LightSensor_StartTask();
   ```

4. **Use the global variables**

   ```c
   extern volatile uint8_t g_handNear;     // 1 = dark/object detected
   extern volatile uint32_t g_lightRaw;    // OPT3001 raw value
   extern volatile uint8_t g_alarmLatched; // 1 = alarm active (latched)
   ```

5. **How to stop the buzzer**

   ```c
   g_alarmLatched = 0;   // clears alarm + stops buzzer
   ```

6. **How the module behaves**

   * When it gets dark or an object is detected â†’
     `g_handNear = 1` and `g_alarmLatched = 1`
   * Buzzer **beeps continuously** while `g_alarmLatched == 1`
   * Buzzer stops only when **your code** sets `g_alarmLatched = 0`
